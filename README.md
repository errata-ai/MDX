# MDX

A Vale configuration for sites using [MDX][1]:

- [x] Adds support for using MDX comment syntax (`{/* ... */}`).
- [x] Ignores `import`s and `export`s.
- [x] Ignores `<Component ... />` and `<Component> ... </Component>`.
- [x] Ignores `{ ... }`.

## Testing

```
$ make test
```

[1]: https://mdxjs.com
